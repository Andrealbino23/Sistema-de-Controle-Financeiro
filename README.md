# Sistema-de-controle-financeiro#  Sistema de Controle Financeiro

Um sistema de controle financeiro desenvolvido com **HTML, CSS e JavaScript**, que permite cadastrar receitas, visualizar o saldo total e acompanhar os registros financeiros de forma simples e intuitiva.

##  Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, como:

* Manipulação do DOM
* Eventos em JavaScript
* Arrays e Objetos
* Funções
* Validação de formulários
* Organização de código
* Lógica de programação

Além disso, futuramente o projeto será evoluído para utilizar **Node.js**, **Express** e **SQLite**, tornando-se uma aplicação Full Stack.

---

##  Funcionalidades

* ✅ Cadastro de receitas
* ✅ Validação dos campos do formulário
* ✅ Exibição dinâmica das receitas cadastradas
* ✅ Cálculo automático do saldo total
* ✅ Interface simples e responsiva
* 🔜 Cadastro de despesas
* 🔜 Exclusão e edição de registros
* 🔜 Pesquisa por descrição
* 🔜 Relatórios mensais
* 🔜 Persistência de dados com SQLite
* 🔜 API REST utilizando Node.js e Express
* 🔜 Dashboard com gráficos

##  Funcionamento

1. O usuário preenche o formulário com os dados da receita.
2. O JavaScript valida as informações.
3. Uma nova receita é criada como um objeto.
4. O objeto é armazenado em um array.
5. A lista de receitas é atualizada automaticamente.
6. O saldo total é recalculado.

Fluxo do sistema:

```text
Usuário
     │
     ▼
Preenche o formulário
     │
     ▼
Clica em "Cadastrar"
     │
     ▼
JavaScript captura os dados
     │
     ▼
Validação
     │
     ▼
Criação do objeto Receita
     │
     ▼
Adiciona ao Array
     │
     ▼
Atualiza a lista
     │
     ▼
Recalcula o saldo
```

## ▶️ Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/controle-financeiro.git
```

2. Acesse a pasta do projeto:

```bash
cd controle-financeiro
```

3. Abra o arquivo `index.html` no navegador.

> Não é necessário instalar dependências nesta versão, pois o projeto é desenvolvido apenas com HTML, CSS e JavaScript.
